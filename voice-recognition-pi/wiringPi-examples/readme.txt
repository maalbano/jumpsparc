These files are used by the jasper modules, but they can also be executed on their own.

To build these files:

1. On your Pi, install wiringPi from http://wiringpi.com/download-and-install/ 
2. copy the files to ~/wiringPi/examples/
3. execute: "$ make beeper" or "$ make FourMinuteCoffee"


To run these files:

1. go to the examples folder.
2. bash $ make beeper
3. bash $ ./beeper
